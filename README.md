# FabricLens AI - Premium Clothing Scanner

**Know Your Clothes Before You Wear Them.**

A premium AI-powered mobile application that analyzes clothing fabric composition and properties.

## Overview

FabricLens AI is a production-ready mobile application for iOS and Android that uses AI to analyze clothing fabric type, composition, and care requirements through image recognition.

## Key Features

- 📸 AI-powered fabric analysis from photos
- 💾 Scan history with search and filtering
- 🌍 Multi-language support (English, Hindi, Arabic)
- 🎨 Premium UI with animations
- 🔐 Secure authentication
- 📊 Detailed fabric properties and recommendations

## Tech Stack

- Mobile: React Native / Flutter
- Backend: Node.js + Express
- Database: MongoDB
- AI: TensorFlow.js
- Auth: Firebase

## Getting Started

1. Clone the repository
2. Follow setup instructions in `docs/SETUP.md`
3. Install dependencies
4. Start development server
