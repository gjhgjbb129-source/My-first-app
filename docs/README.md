# FabricLens AI

**Know Your Clothes Before You Wear Them.**

A premium AI-powered mobile application that analyzes clothing fabric using advanced image recognition.

## Features

✨ **Premium UI** - Material Design 3, Glassmorphism, smooth animations
🤖 **AI Analysis** - Fabric detection and properties analysis
📸 **Camera Integration** - High-quality photo capture
🌍 **Multi-language** - English, Hindi, Arabic support
🎨 **Theme Support** - Light, Dark, System themes
📱 **Cross-platform** - iOS and Android

## Setup

See `docs/SETUP.md` for detailed installation instructions.

## Tech Stack

- **Mobile**: React Native with TypeScript
- **Backend**: Node.js with Express
- **Database**: MongoDB
- **Authentication**: Firebase
- **ML**: TensorFlow.js
